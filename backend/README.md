# kip-system

КИМ система для проекта СамГТУ

## Устновка приложения

```bash
cd shell-scripts
sudo chmod u+x install.sh
./install.sh
```

Установщик настроит python окружение, а также создаст daemon (Ubuntu 22.04).

## Запуск приложения

```bash
systemctl start kip-system.service
```
